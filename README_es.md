# Cómo Usar GPT Image 2 para E-commerce

> Un recetario de prompts para crear imágenes de e-commerce con **GPT Image 2** — fotos de listado, prueba con modelos, presentaciones de producto, escenas de interacción y creatividades de social commerce.

## 🍌 Introducción

Este repositorio te muestra **cómo usar GPT Image 2 para crear imágenes de e-commerce** — fotos de listado, imágenes principales para múltiples países, visuales de detalle A+, prueba con modelos, presentaciones de producto, escenas de interacción y creatividades de social commerce — a escala de producción.

**Cada caso es una receta:** una foto del producto de entrada, el prompt exacto para pegar en GPT Image 2 y ejemplos reales de salida. Reemplaza los `{placeholders}` con tu producto, idioma, país y puntos de venta.

¿Necesitas procesar cientos de SKU a la vez? Salta a [🚀 Generación en Lote vía API de Evolink](#-generación-en-lote-vía-api-de-evolink).

Pruébalo en Evolink: [Flujo de trabajo GPT Image 2](https://evolink.ai?utm_source=github&utm_medium=readme&utm_campaign=awesome-ecommerce-image-prompts)

Si te resulta útil, ¡danos una Star ⭐!

> [!NOTE]
> Este repositorio se enfoca en plantillas de prompts reutilizables para creatividades de e-commerce. Reemplaza los placeholders entre llaves antes de usar y ajústalos a tu plataforma, mercado, idioma, producto, material, tipo de modelo y puntos de venta reales.

1. **Elige un caso** abajo que coincida con tu objetivo de e-commerce (imagen principal, prueba con modelo, producto multi-ángulo, visuales A+, etc.).
2. **Sube la foto de tu producto** y **copia el prompt** de ese caso — reemplaza los `{placeholders}` con tu producto, idioma, país, plataforma y puntos de venta.

## 🖼️ Prompts de Imágenes para Listados de E-commerce

### Caso 1: Set de Imágenes para Marketplace

**Prompt:**

```
Generate a {platform} listing image set for {country} in {language}. The product is {product_name}, and the selling points are {selling_points}.
```

> [!NOTE]
> **Reemplaza {platform}, {country}, {language}, {product_name} y {selling_points} antes de usar.** Esta plantilla se adapta a Amazon, eBay, AliExpress, TEMU, SHEIN, TikTok Shop, Shopee, Lazada, Mercado Libre, Walmart, Etsy, Rakuten, Coupang, Shopify y marketplaces similares. El agente planificará activos de listado apropiados según la plataforma: imágenes con fondo blanco limpio, diagramas de tamaño, vistas multi-ángulo, escenas lifestyle, imágenes de puntos de venta y detalles de materiales.

### Caso 4: Generación Masiva de Listados de Amazon

## 🚀 Generación en Lote vía API de Evolink

Para ejecuciones de producción a través de cientos de SKU, múltiples idiomas o varios marketplaces, llama directamente a la **API de GPT Image 2 de Evolink** en vez de ejecutar prompts uno a uno.

**Ventajas del procesamiento por lotes vía API:**

- Generación masiva de imágenes de listado para todo el catálogo
- Traducción multilingüe de todos los activos de listado en una sola tarea
- Cambio de modelo, reemplazo de fondo y salida multi-ángulo programáticos
- Callbacks webhook se integran con tu PIM, ERP o automatización de marketing

**Requisitos previos:** [Obtén una clave API](https://evolink.ai/dashboard/keys).

### Paso 1: Enviar una tarea de generación

La generación de GPT Image 2 es **asíncrona** — la llamada devuelve un `task_id` que consultas para obtener resultados.

```bash
curl -X POST https://api.evolink.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "Generate an Amazon listing image set for the US in English. The product is a solid wood dining chair, and the selling points are sturdy frame, ergonomic backrest, easy assembly.",
    "image_urls": ["https://your-cdn.example.com/products/chair.jpg"],
    "size": "1:1",
    "resolution": "2K",
    "quality": "high",
    "n": 4
  }'
```

| Campo | Requerido | Notas |
| :--- | :--- | :--- |
| `model` | sí | Debe ser `"gpt-image-2"` |
| `prompt` | sí | Hasta 32,000 caracteres — pega el prompt de cualquier caso de este repo |
| `image_urls` | opcional | 1–16 imágenes de entrada/referencia (jpeg/png/webp, ≤50 MB cada una) para tareas image-to-image y edición |
| `size` | opcional | Relación de aspecto (`1:1`, `2:3`, `16:9`, …) o dimensiones en píxeles; por defecto `auto` |
| `resolution` | opcional | `1K`, `2K` o `4K` cuando `size` es una relación |
| `quality` | opcional | `low`, `medium`, `high` — afecta al coste; por defecto `medium` |
| `n` | opcional | 1–10 imágenes por llamada |
| `callback_url` | opcional | Webhook HTTPS para recibir completado en lugar de hacer polling |

### Paso 2: Consultar la tarea

```bash
curl https://api.evolink.ai/v1/tasks/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Cuando `status` pasa a `"completed"`, el array `results` contiene las URLs de las imágenes generadas. **Las imágenes generadas son válidas durante 24 horas — descárgalas con prontitud.**

### Opcional: callback webhook

Pasa `"callback_url": "https://your-server.example.com/webhook"` en el envío y Evolink hará POST con el resultado a tu endpoint cuando la tarea se complete — sin polling.

**Referencia completa de la API:** [docs.evolink.ai · Generación de imágenes GPT Image 2](https://docs.evolink.ai/en/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation)
