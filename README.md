## 🍌 Introduction

Welcome to the ecommerce image generation prompt repository! 🤗

**We collect high-quality prompts and workflow patterns for ecommerce listing images, model try-ons, product showcases, interactive scenes, short-form sales videos, and social commerce content.**

This repository turns practical ecommerce creative production knowledge into reusable English-first prompt templates for sellers, designers, cross-border teams, and growth operators working across marketplaces and languages.

## 🖼️ Ecommerce Listing Image Prompts

<!-- Case 1: Marketplace Listing Image Set -->
### Case 1: Marketplace Listing Image Set

| Output |
| :----: |
| <img src="images/ecommerce_listing_images_case1/output.jpg" width="300" alt="Output image"> |

**Prompt:**

```
Generate a full {platform} listing image set for the {country} market in {language}. The product is {product_name}. Highlight these selling points: {selling_points}. Plan the image set according to the platform's listing logic and include a clean white-background hero image, size chart, multi-angle product views, lifestyle usage scenes, key benefit slides, and material detail close-ups.
```

> [!NOTE]
> **Replace {platform}, {country}, {language}, {product_name}, and {selling_points} before use.** This template can be adapted for Amazon, eBay, AliExpress, TEMU, SHEIN, TikTok Shop, Shopee, Lazada, Mercado Libre, Walmart, Etsy, Rakuten, Coupang, Shopify, and other ecommerce platforms.

<!-- Case 2: Multi-Country Listing Image Set -->
### Case 2: Multi-Country Listing Image Set

| Output |
| :----: |
| <img src="images/ecommerce_listing_images_case2/output.jpg" width="300" alt="Output image"> |

**Prompt:**

```
Generate a complete listing image set for {platform}, targeting {country}, in {language}.
```

> [!NOTE]
> **Replace {platform}, {country}, and {language} before use.** Use this shorter template when you want the model to infer the best image structure for a marketplace automatically.

<!-- Case 3: A+ Product Detail Visuals -->
### Case 3: A+ Product Detail Visuals

**Prompt:**


> [!NOTE]
> **Replace {product_description}, {country}, and {language} before use.** This pattern works well for product storytelling, feature breakdowns, usage benefits, and premium detail layouts.

<!-- Case 4: Bulk Product Main Image Replacement -->
### Case 4: Bulk Product Main Image Replacement

| Input | Output |
| :---: | :----: |
| <img src="images/ecommerce_listing_images_case4/input.jpg" width="300" alt="Input image"> | <img src="images/ecommerce_listing_images_case4/output.jpg" width="300" alt="Output image"> |

**Input:** Upload the source product image and the target listing image set.

**Prompt:**

```
Replace the product from image 1 into images 2 through 7. Only change the product itself and keep all other layout, lighting, text, and scene elements unchanged.
```

> [!NOTE]
> This template is designed for one-click bestseller replication across multiple SKUs while keeping the creative structure consistent.

<!-- Case 5: Bulk Listing Translation -->
### Case 5: Bulk Listing Translation

| Input | Output |
| :---: | :----: |
| <img src="images/ecommerce_listing_images_case5/input.jpg" width="300" alt="Input image"> | <img src="images/ecommerce_listing_images_case5/output.jpg" width="300" alt="Output image"> |

**Input:** Upload all listing images that contain product text.

**Prompt:**

```
Translate all text across the uploaded images into {target_language} while preserving the original design, spacing, layout, hierarchy, and composition.
```

> [!NOTE]
> **Replace {target_language} before use.** This is useful for global rollout when you want to keep the same visual system across different markets.

## 👗 Model Try-On and Apparel Prompts

<!-- Case 1: Apparel Try-On Image Set -->
### Case 1: Apparel Try-On Image Set

| Output |
| :----: |
| <img src="images/model_try_on_and_apparel_case1/output.jpg" width="300" alt="Output image"> |

**Prompt:**

```
This is a {garment_type}. Generate a complete try-on image set for selling it in {country}.
```

> [!NOTE]
> **Replace {garment_type} and {country} before use.** Use this for dresses, tops, outerwear, children’s clothing, or any apparel category that needs a commercial try-on presentation.

<!-- Case 2: Model Swap with Appearance Control -->
### Case 2: Model Swap with Appearance Control

| Input | Output |
| :---: | :----: |
| <img src="images/model_try_on_and_apparel_case2/input.jpg" width="300" alt="Input image"> | <img src="images/model_try_on_and_apparel_case2/output.jpg" width="300" alt="Output image"> |

**Input:** Upload the original product-on-model image.

**Prompt:**

```
Replace the model with a {ethnicity} model, {hairstyle_description}, while keeping the clothing unchanged.
Replace the person with a {ethnicity} model, {skin_tone}, {hair_color}, while keeping the outfit unchanged.
```

> [!NOTE]
> **Replace {ethnicity}, {hairstyle_description}, {skin_tone}, and {hair_color} before use.** This template helps localize fashion creatives for different audiences without re-shooting garments.

<!-- Case 3: Background Concepts for Apparel Sales -->
### Case 3: Background Concepts for Apparel Sales

| Output |
| :----: |
| <img src="images/model_try_on_and_apparel_case3/output.jpg" width="300" alt="Output image"> |

**Prompt:**

```
Generate suitable background concepts for selling {garment_type}. Design several indoor and outdoor background options for me to choose from.
```

> [!NOTE]
> **Replace {garment_type} before use.** This works well when you want to separate scene ideation from model generation.

<!-- Case 4: Accessories, Nails, and Shoes Try-On -->
### Case 4: Accessories, Nails, and Shoes Try-On

| Output |
| :----: |
| <img src="images/model_try_on_and_apparel_case4/output.jpg" width="300" alt="Output image"> |

**Prompt:**

```
A model is wearing this {accessory_type}, {additional_requirements}.
A model is wearing the shoes from the image, styled with {outfit_description}.
Apply these nails to a realistic hand presentation.
```

> [!NOTE]
> **Replace {accessory_type}, {additional_requirements}, and {outfit_description} before use.** You can use this single pattern family for jewelry, earrings, nails, and footwear try-on concepts.
