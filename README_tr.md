1. **E-ticaret hedefinize uygun vakayı seçin** (ana liste görseli, model prova, çok açılı ürün, A+ detay görselleri vb.).
2. **Ürün fotoğrafınızı yükleyin** ve o vakanın **promptunu kopyalayın** — `{placeholders}` yerine kendi ürün, dil, ülke, platform ve satış noktalarınızı yazın.
3. **GPT Image 2 üzerinde çalıştırın** — tek seferlik kreatifler için [Evolink iş akışı UI](https://evolink.ai/gpt-image-2?utm_source=github&utm_medium=readme&utm_campaign=gpt-image-2-for-e-commerce, toplu SKU ve çok dilli yayılımlar için [Evolink API](#-evolink-api-ile-toplu-üretim) üzerinden.

## 🖼️ E-ticaret Liste Görseli Promptları

### Vaka 1: Marketplace Liste Görsel Seti

**Prompt:**

```
Generate a {platform} listing image set for {country} in {language}. The product is {product_name}, and the selling points are {selling_points}.
```

> [!NOTE]
> **Kullanmadan önce {platform}, {country}, {language}, {product_name} ve {selling_points} alanlarını değiştirin.** Bu şablon Amazon, eBay, AliExpress, TEMU, SHEIN, TikTok Shop, Shopee, Lazada, Mercado Libre, Walmart, Etsy, Rakuten, Coupang, Shopify ve benzeri marketplace'lere uygundur. Agent, platforma uygun liste varlıklarını planlar: temiz beyaz arka plan görselleri, boyut diyagramları, çok açılı görünümler, yaşam tarzı sahneleri, satış noktası görselleri ve malzeme detayları.

## 🚀 Evolink API ile Toplu Üretim

Yüzlerce SKU, birden fazla dil veya birden fazla marketplace üzerindeki üretim koşuları için promptları tek tek çalıştırmak yerine doğrudan **Evolink GPT Image 2 API'sini** çağırın.

**Neden API ile toplu işlem:**

- Tüm katalog için toplu liste görseli üretimi
- Tüm liste varlıklarının çok dilli çevirisi tek bir görevde
- Programatik model değişimi, arka plan değiştirme, çok açılı çıktı
- Webhook callback'leri PIM, ERP veya pazarlama otomasyonunuzla entegre olur

**Ön koşullar:** [API anahtarı alın](https://evolink.ai/dashboard/keys).

### Adım 1: Üretim görevi gönderme

GPT Image 2 üretimi **asenkrondur** — çağrı bir `task_id` döndürür ve sonuçları sorgulayarak alırsınız.

```bash
curl -X POST https://api.evolink.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "Generate an Amazon listing image set for the US in English. The product is a solid wood dining chair, and the selling points are sturdy frame, ergonomic backrest, easy assembly.",
    "image_urls": ["https://your-cdn.example.com/products/chair.jpg"],
    "size": "1:1",
    "resolution": "2K",
    "quality": "high",
    "n": 4
  }'
```

| Alan | Zorunlu | Notlar |
| :--- | :--- | :--- |
| `model` | evet | `"gpt-image-2"` olmalı |
| `prompt` | evet | 32.000 karaktere kadar — bu repo'daki herhangi bir vakanın promptunu yapıştırın |
| `image_urls` | opsiyonel | image-to-image ve düzenleme görevleri için 1–16 giriş/referans görsel (jpeg/png/webp, her biri ≤50 MB) |
| `size` | opsiyonel | En-boy oranı (`1:1`, `2:3`, `16:9`, …) veya piksel boyutu; varsayılan `auto` |
| `resolution` | opsiyonel | `size` bir oran ise `1K`, `2K` veya `4K` |
| `quality` | opsiyonel | `low`, `medium`, `high` — maliyeti etkiler; varsayılan `medium` |
| `n` | opsiyonel | Çağrı başına 1–10 görsel |
| `callback_url` | opsiyonel | Polling yerine tamamlanma bildirimi alacak HTTPS webhook |

### Adım 2: Görevi sorgulama

```bash
curl https://api.evolink.ai/v1/tasks/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

`status` `"completed"` olduğunda, `results` dizisi üretilen görsel URL'lerini içerir. **Üretilen görseller 24 saat geçerlidir — gecikmeden indirin.**

### Opsiyonel: webhook callback

Gönderimde `"callback_url": "https://your-server.example.com/webhook"` geçirin; görev tamamlandığında Evolink sonucu endpoint'inize POST eder — polling gerekmez.

**Tam API referansı:** [docs.evolink.ai · GPT Image 2 görsel üretimi](https://docs.evolink.ai/en/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation)
