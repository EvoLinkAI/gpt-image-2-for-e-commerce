# Como Usar GPT Image 2 para E-commerce

> Um livro de receitas de prompts para criar imagens de e-commerce com **GPT Image 2** — fotos de listagem, prova de roupa em modelo, showcases de produto, cenas de interação e criativos de social commerce.

## 🍌 Introdução

Este repositório te mostra **como usar GPT Image 2 para criar imagens de e-commerce** — fotos de listagem, imagens principais multi-país, visuais de detalhe A+, prova em modelo, showcases de produto, cenas de interação e criativos de social commerce — em escala de produção.

**Cada caso é uma receita:** uma foto do produto de entrada, o prompt exato para colar no GPT Image 2 e exemplos reais de saída. Substitua os `{placeholders}` pelo seu produto, idioma, país e pontos de venda.

Precisa processar centenas de SKUs de uma vez? Pule para [🚀 Geração em Lote via API Evolink](#-geração-em-lote-via-api-evolink).

Experimente no Evolink: [Workflow GPT Image 2](https://evolink.ai/gpt-image-2?utm_source=github&utm_medium=readme&utm_campaign=gpt-image-2-for-e-commerce)

Se for útil, deixe uma Star ⭐

1. **Escolha um caso** abaixo que corresponda ao seu objetivo de e-commerce (imagem principal, prova em modelo, produto multi-ângulo, visuais A+, etc.).
2. **Faça upload da foto do seu produto** e **copie o prompt** do caso — substitua os `{placeholders}` pelo seu produto, idioma, país, plataforma e pontos de venda.
3. **Execute no GPT Image 2** — seja pela [UI do workflow Evolink](https://evolink.ai/gpt-image-2?utm_source=github&utm_medium=readme&utm_campaign=gpt-image-2-for-e-commerce) para criativos pontuais, ou via [API Evolink](#-geração-em-lote-via-api-evolink) para SKUs em massa e rollouts multilíngues.

## 🖼️ Prompts de Imagens de Listagem E-commerce

### Caso 1: Set de Imagens para Marketplace

**Prompt:**

```
Generate a {platform} listing image set for {country} in {language}. The product is {product_name}, and the selling points are {selling_points}.
```

> [!NOTE]
> **Substitua {platform}, {country}, {language}, {product_name} e {selling_points} antes de usar.** Este template se adapta a Amazon, eBay, AliExpress, TEMU, SHEIN, TikTok Shop, Shopee, Lazada, Mercado Libre, Walmart, Etsy, Rakuten, Coupang, Shopify e marketplaces similares. O agente planejará ativos de listagem apropriados para a plataforma, como imagens com fundo branco limpo, diagramas de tamanho, vistas multi-ângulo, cenas lifestyle, imagens de pontos de venda e detalhes de material.

## 🚀 Geração em Lote via API Evolink

Para execuções de produção em centenas de SKUs, vários idiomas ou vários marketplaces, chame diretamente a **API GPT Image 2 da Evolink** em vez de rodar prompts um a um.

**Por que processar em lote via API:**

- Geração em massa de imagens de listagem para um catálogo inteiro
- Tradução multilíngue de todos os ativos de listagem em uma única tarefa
- Troca de modelo, substituição de fundo e saída multi-ângulo programáticas
- Callbacks webhook integram com seu PIM, ERP ou automação de marketing

**Pré-requisitos:** [Obtenha uma chave API](https://evolink.ai/dashboard/keys).

### Passo 1: Enviar uma tarefa de geração

A geração GPT Image 2 é **assíncrona** — a chamada retorna um `task_id` que você consulta para obter os resultados.

```bash
curl -X POST https://api.evolink.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "Generate an Amazon listing image set for the US in English. The product is a solid wood dining chair, and the selling points are sturdy frame, ergonomic backrest, easy assembly.",
    "image_urls": ["https://your-cdn.example.com/products/chair.jpg"],
    "size": "1:1",
    "resolution": "2K",
    "quality": "high",
    "n": 4
  }'
```

| Campo | Obrigatório | Notas |
| :--- | :--- | :--- |
| `model` | sim | Deve ser `"gpt-image-2"` |
| `prompt` | sim | Até 32.000 caracteres — cole o prompt de qualquer caso deste repo |
| `image_urls` | opcional | 1–16 imagens de entrada/referência (jpeg/png/webp, ≤50 MB cada) para tarefas image-to-image e edição |
| `size` | opcional | Proporção (`1:1`, `2:3`, `16:9`, …) ou dimensões em pixels; padrão `auto` |
| `resolution` | opcional | `1K`, `2K` ou `4K` quando `size` é uma proporção |
| `quality` | opcional | `low`, `medium`, `high` — afeta o custo; padrão `medium` |
| `n` | opcional | 1–10 imagens por chamada |
| `callback_url` | opcional | Webhook HTTPS para receber conclusão em vez de polling |

### Passo 2: Consultar a tarefa

```bash
curl https://api.evolink.ai/v1/tasks/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

Quando `status` se torna `"completed"`, o array `results` contém as URLs das imagens geradas. **Imagens geradas são válidas por 24 horas — baixe-as prontamente.**

### Opcional: callback webhook

Passe `"callback_url": "https://your-server.example.com/webhook"` no envio e a Evolink fará POST do resultado para seu endpoint quando a tarefa concluir — sem polling.

**Referência API completa:** [docs.evolink.ai · Geração de imagens GPT Image 2](https://docs.evolink.ai/en/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation)
