# 如何使用 GPT Image 2 製作電商素材

> 一份面向 **GPT Image 2** 的電商素材製作提示詞手冊——涵蓋主圖套圖、多國主圖、模特試穿、產品展示、互動場景、社交電商等場景。

## 🍌 簡介

本倉庫教你**如何使用 GPT Image 2 製作電商素材**——電商主圖、多國本地化套圖、A+ 產品詳情、模特試穿、產品多角度展示、互動場景、社交電商帶貨圖——並支援規模化生產。

**每個案例都是一份配方：** 一張產品原圖 + 直接複製即用的 GPT Image 2 提示詞 + 真實輸出示例。使用前把 `{佔位符}` 替換為你自己的商品、語言、國家和賣點即可。

## 🖼️ 電商主圖提示詞

### 案例 1：電商平臺主圖套圖

**提示詞：**

```
幫我生成一組{平臺名}套圖，售賣到{國家}，{語言}，這款商品為{商品名}，賣點是{賣點描述}
```

> [!NOTE]
> **使用前請替換 {平臺名}、{國家}、{語言}、{商品名} 和 {賣點描述}。** 此模板適用於 Amazon、eBay、AliExpress、TEMU、SHEIN、TikTok Shop、Shopee、Lazada、Mercado Libre、Walmart、Etsy、Rakuten、Coupang、Shopify 等電商平臺。Agent 會自動規劃商品適合的套圖內容及平臺特點，生成包括：白底圖、尺寸圖、多角度展示圖、使用場景圖、賣點圖、材質細節圖等。

## 🚀 批次生成 · 呼叫 Evolink API

如果你需要在大量 SKU、多語言、多平臺之間規模化生產電商素材，建議直接呼叫 **Evolink GPT Image 2 API**，而不是逐張手動生成。

**為什麼用 API 跑批次：**

- 整盤 SKU 的電商主圖一次性出圖
- 一次任務搞定全部主圖的多語言版本翻譯
- 程式化模特替換、背景替換、多角度出圖
- Webhook 回撥可直接對接 PIM、ERP 或營銷自動化系統

**前置條件：** [獲取 API Key](https://evolink.ai/dashboard/keys)。

### 步驟 1：提交生成任務

GPT Image 2 是**非同步任務**——介面返回 `task_id`，需要輪詢任務詳情拿到結果。

```bash
curl -X POST https://api.evolink.ai/v1/images/generations \
  -H "Authorization: Bearer YOUR_API_KEY" \
  -H "Content-Type: application/json" \
  -d '{
    "model": "gpt-image-2",
    "prompt": "幫我生成一組Amazon套圖，售賣到美國，英文，這款商品為實木餐椅，賣點是穩固框架、人體工學椅背、易組裝",
    "image_urls": ["https://your-cdn.example.com/products/chair.jpg"],
    "size": "1:1",
    "resolution": "2K",
    "quality": "high",
    "n": 4
  }'
```

| 欄位 | 是否必填 | 說明 |
| :--- | :--- | :--- |
| `model` | 必填 | 固定為 `"gpt-image-2"` |
| `prompt` | 必填 | 最長 32,000 字元——直接複製本倉庫任意 case 的提示詞即可 |
| `image_urls` | 可選 | 1–16 張參考圖（jpeg/png/webp，單張 ≤50 MB），用於圖生圖與編輯場景 |
| `size` | 可選 | 比例（`1:1`、`2:3`、`16:9` …）或畫素尺寸；預設 `auto` |
| `resolution` | 可選 | `1K`、`2K`、`4K`，配合比例使用 |
| `quality` | 可選 | `low` / `medium` / `high`，影響計費；預設 `medium` |
| `n` | 可選 | 單次生成 1–10 張 |
| `callback_url` | 可選 | HTTPS Webhook 地址，任務完成後回撥，無需輪詢 |

### 步驟 2：輪詢任務結果

```bash
curl https://api.evolink.ai/v1/tasks/{task_id} \
  -H "Authorization: Bearer YOUR_API_KEY"
```

當 `status` 變為 `"completed"` 時，`results` 數組裡就是生成的圖片 URL。**生成的圖片僅 24 小時內有效，請儘快下載留存。**

### 可選：Webhook 回撥

在提交任務時傳入 `"callback_url": "https://your-server.example.com/webhook"`，Evolink 會在任務完成後 POST 結果到你指定的介面，無需輪詢。

**完整 API 文件：** [docs.evolink.ai · GPT Image 2 image generation](https://docs.evolink.ai/en/api-manual/image-series/gpt-image-2/gpt-image-2-image-generation)
